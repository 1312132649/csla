Hi All,

I get the strangest error when I try to add a new Data source to my UI project. I click on Add new Data Source in the Data source window, the datasource configuration wizard pops up, I select Object, click Next and then I get an Object reference not set to an instance of an object error.

Thanks!


Pieterv replied on Thursday, August 30, 2007

Hi All,

I've found the problem. The UI project must have a class in it's root. I've structured my project into folders, thus there was no class in the root, creating a blank class1.vb in the root of the project solved the problem.

Go Figure!

RockfordLhotka posted on Friday, November 01, 2013

I have put a new beta release online via nuget and installer.

This prerelease includes some key elements:

  • Primary solution converted to Visual Studio 2013
  • WinRT projects now target WinRT 8.1
  • Csla.Xaml for WinRT now includes full PropertyInfo control like in WPF/SL
  • Android support (via Xamarin tools)
  • Csla.Axml for Android includes useful Android UI helper types
  • EF 6 support
  • Dropped MVC 3 support
  • Some samples updated
  • Bug fixes
  • Release version is now 3 digits to allow for maintenance of previous releases

At this point we're resolving some final issues with nuget, and are updating more of the samples. No more major changes are expected as we head toward the 4.5.500 release.

If you are using 4.5.40 please download this version and let us know if you encounter any issues.

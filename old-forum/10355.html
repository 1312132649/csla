Can we use BusinessBase.Clone()  ?



RockfordLhotka replied on Friday, May 13, 2011

A clone is an exact copy of the original, including all private field values, and all child objects - it is a copy of the entire object graph.

The Clone method uses the same technology the data portal uses to move the object graph between the client and application server.

So yes, you can use Clone to create an exact copy of the original object graph.

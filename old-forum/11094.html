RockfordLhotka replied on Thursday, January 26, 2012

I do not plan to write a Web Forms book at this time, no.

This is mostly because there've been few meaningful changes to Web Forms since 2005, and no changes to how CSLA supports Web Forms for many years. It was 2009 when I split the assemblies apart, but even that didn't change the functionality at all. I think the last time the Csla.Web functionality changed was in 2006...

What this means is that the chapter in the Expert 2008 Business Objects book on Web Forms development is still 100% relevant.

But there are a bunch of changes coming in .NET 4.5, and I suspect CSLA 4 version 4.5 will require some new features to support the new Web Forms capabiltiies that are coming. From what I've seen, these new Web Forms features should be pretty cool, and that'll almost certainl require new book or video content to document how to use them with CSLA 4 version 4.5.

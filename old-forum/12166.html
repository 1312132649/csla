RockfordLhotka posted on Friday, September 27, 2013

Hello everyone,

I have release CSLA 4 version 4.5.40 via nuget.

The primary focus of this release is updated Android support that uses the latest Xamarin tools to provide parity with the other platforms we support, including the async/await keywords. Kevin has more info about Android here:

http://windingroadway.blogspot.com/2013/09/csla-xamarin-android.html

There are also lots of bug fixes and other smaller feature improvements, making this an important release for anyone using 4.5. The changes are listed here:

https://github.com/MarimerLLC/csla/issues?milestone=6&page=1&state=closed

The next release (4.5.50) will focus on Windows 8.1, and should come out relatively soon if my health holds. I'd already done a lot of the work to bring the Csla.Xaml controls into line with WPF/SL, so most of the remaining work in 4.5.50 is stabilization, testing, and completeness.

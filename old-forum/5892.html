AndyW posted on Thursday, November 27, 2008

 

Hi,

I am passing  my business object to a form like this,  how do I pass a new object?

using (Forms.ArticleKeyword aKey = new Forms.ArticleKeyword((Business.ArticleKeyword)articleKeywords.Current))
{
      aKey.ShowDialog();
}

 

Thanks

danielmartind replied on Thursday, November 27, 2008

Hi,

Something like

using (Forms.ArticleKeyword aKey = new Forms.ArticleKeyword(Business.ArticleKeyword.NewArticleKeyword()))
{
      aKey.ShowDialog();
}

or if you overide AddNewCore in your collection class (Asuming it is a collection you are using)

using (Forms.ArticleKeyword aKey = new Forms.ArticleKeyword((Business.ArticleKeyword)articleKeywords.AddNew()))
{
      aKey.ShowDialog();
}

Dan

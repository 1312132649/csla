RockfordLhotka posted on Wednesday, August 11, 2010

CSLA 4 version 4.0.1 Beta 1 is now available for download.

This is a bug fix release. It includes two breaking changes:

  • You must now use the PrivateField flag when calling RegisterProperty() if you are using a private backing field
  • Exceptions thrown from Save() and BeginSave() may be different

The CSLA .NET license has been tweaked slightly to clarify some wording (based on some legal feedback). This has no effect on the intent, scope or effect of the license.

The full source download includes my work in progress code for Windows Phone 7. That work is not complete, and I'm not sure you can actually do anything with the current code, though many unit tests do pass. You have been warned!

Otherwise see the change log for a list of bugs that have been fixed.

    Hi,all,
    I'm running a c# experimented framework leveraging Csla.Net WITH NHibernate.
   
    My goal is,with this framework we can build flexible and scalable LOB(Line of business) applications more rapidly basing on Csla.Net and NHibernate.
 
    The framework was born from a project called 'cslaptrackerfactory',which was developped by some pioneers as a drop in replacement for the standard Csla ProjectTracker.Library.dll.
   
    It uses NHibernate as the ORM on all DAL tasks,and Csla.net to manipulate the business layer.

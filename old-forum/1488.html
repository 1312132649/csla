Brian Criswell posted on Saturday, October 14, 2006

I have published a new version of the ObjectListView to the CSLAcontrib project on CodePlex.  Significant new features include:

  • Added sorting and filtering of arrays
  • Added sorting and filtering of an IList that contains a primitive data type (string, int, enum, etc.)
  • Added ability to set the List property outside of the constructor.
  • Added visual designer support (although it is of limited use)

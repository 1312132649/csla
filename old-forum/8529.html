Code Complete 4.2 has been released and is available for download. The new version supports the latest version of CSLA.NET (3.8.2), includes some bug fixes and the following improvements:

  1. The previous evaluation copy of Code Complete limits the number of columns for each table to 5.  This limitation has been removed and the evaluation copy is now fully functional. You can generate fully working Windows, Web, WPF and Silverlight for free.
  2. In previous versions, Code Complete generates web sites. Version 4.2 supports both the generation of web sites and web applications. By default, Code Complete now generates Web Applications. You can change using the Options dialog accessible from the Web Application Project tab in the solution wizard.
  3. Version 4.2 includes improvements for Firebird. Code Complete now supports Boolean and GUID data types and Identity and Timestamp fields. A guide on how to use these features in Firebird will be released soon.

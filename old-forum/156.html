RockfordLhotka replied on Saturday, May 20, 2006

You might be able to get it to work with people in the Power User role, but no lower (to my knowledge).

To get it to work with the Power User role, change NetRun so it alters the permission set at the user's level rather than at the machine level. A Power User (or so I've been told) should have the rights to alter the permission sets at the user level.

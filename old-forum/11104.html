Nightly Build 2012.01.29 is a fix release but includes a major UI feature: the status bar is used to display useful information and gives you access to the lists of errors and warnings of the last run of code generation.

New Features (since Nightly Build 2012.01.15 - DeepData for Encapsulated Implementation):

  • Improve status bar and error/warning report
  • List Csla Objetcs - add key accelerators and "Select All" menu item
  • Generate Contains and ContainsDeleted methods with object key properties

Plus other improvements and fixes.

Don't forget that you need full trust. May be that's what you mean by full control. Shared hosts always (or at least should) run medium trust so you can't run CSLA. A virtual private server is probably the best option. You can get a low cost Windows VPS at http://www.jodohost.com although I have not tried one so I don't know how good they are. I'm happy with Jodo's shared hosting. Their VPSs come with access to SQL Server on another machine.

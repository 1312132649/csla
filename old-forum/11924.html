A beta version of CSLA 4.5 (version 4.5.22) is now available via nuget.

I am following the approach taken by an increasing number of Microsoft open source projects (like Entity Framework) and only doing pre-releases via nuget. That is much simpler for me, and allows for a more iterative release process.

You can see the list of changes here:

https://github.com/MarimerLLC/csla/issues?labels=&milestone=4&page=1&state=closed

As you can see, the switch to github is turning out to be very valuable, as more people are feeling comfortable submitting pull requests (fixes/features) to the project. That helps our velocity overall, and I really appreciate the help from our newest contributors. You can see a list of team members here:

https://github.com/MarimerLLC?tab=members

The CSLAcontrib project web site is hosted at CodePlex:

RockfordLhotka replied on Sunday, July 16, 2006

If you have developed something that you'd like to contribute to the broader CSLA .NET community, like a code generator, templates, a data portal channel or whatever, you need to register with www.codeplex.com and give me your username. That's what I need to make you a developer in the project.

You'll also need Visual Studio Team System, or need to download the Team Explorer add-in for other versions of VS - that's what you need to interact with the source code repository. There are instructions in the other post I put here with the message from the CodePlex team.

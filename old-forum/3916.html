RockfordLhotka replied on Tuesday, November 20, 2007

Of course I would like everyone to buy several copies of each

But the reality is that you probably only need the 2.1 ebook, if you can figure out the data binding code in the 3.0 version of PTWin yourself.

The 3.0 ebook does include a discussion about the PTWin data binding code and how/why it works. But the majority of the 3.0 ebook is focused on WPF, WCF and WF. So if you can figure out the PTWin code (by looking at the code and/or searching through the forum for the 2-3 threads that discuss the topic), then you probably don't need the 3.0 ebook right now.

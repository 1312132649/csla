The new version of CslaContrib library is available on Codeplex and on NuGet

Changes since last release:

  • Upgrade to Csla 4.5.601
  • Fixes missing WinRT assemblies for CslaContrib.MEF package
  • Visual WebGUI assemblies target v.10.0 and use modal dialogs

 

Note - The release also includes the SmartDateExtendedParser that improves on the standard CSLA SmartDate parser (eg. "+90" will add 90 days to the current date and "0525" converts to "May 25 2014").

I intend to shortly release a project I've been working on as open source, providing a download from my website. This project uses the csla library, of course, which is contained in my VS solution.  Thanks.

RockfordLhotka replied on Sunday, July 08, 2007

You can include the source as long as the license and copyright materials are kept intact.

I don't recommend including the source however, because it will very likely lead to confusion over time (due to the way search engines index these things). Instead, I recommend you reference www.lhotka.net/cslanet or www.lhotka.net/cslanet/download.aspx and specify the version against which you know your code will run.

So the license doesn't preclude you from including the source (with the noted constraints), but there are some very good reasons why it is a terrible, terrible idea, and so I request that you avoid including the source.

Thanks!

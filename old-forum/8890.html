Just to inform you, CslaExtension Beta1 for EF is available for download.

http://t4csla.codeplex.com/

CslaExtension is a Visual Studio extension that adds custom properties to a EF model, and a custom T4 template that generates CSLA classes. 

Key features:

- generation of CSLA classes from EF model

- properties

- business rules

- relations / navigation properties

- collections

- many things to come...

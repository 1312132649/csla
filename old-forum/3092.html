There have been a number of posts about the Smart Client Software Factory (SCSF) and the Composite Applicaton Block (CAB). I know that a number of us have been using the SCSF and CSLA successfully for some time. Now that the latest version of the SCSF has come out, I thought it would be amusing to convert the Project Tracker WinForm example to use the SCSF so that I could learn about its new features.

I have published the details on CodeProject for anyone who is interested and it is available here. This may help anyone who is thinking of using this collection of technologies.

Experience developers here will have a good laugh at the amount of overhead that this approach gives rise to. Equally it does demonstrate the power of a composite approach.

If anyone is interested, you can see how I have used CSLA and the earlier versions of SCSF in my application here.

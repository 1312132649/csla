Ahh, it is a security issue.

 

When you download a zip file Windows notes that it came from the web. When you extract the contents of the zip file Windows notes that the contents came from the web. When you try to open the chm you hit a security barrier because Windows doesn’t trust the data in the file.

 

Right-click on the zip or chm and choose Properties. Then click the Unblock button near the bottom to tell Windows that this file is actually OK to trust.

sergeyb replied on Thursday, April 10, 2008

I never used ADAM, but you are correct I believe.  Simply put your code into YourCustomPrincipal.Login method and have it populate roles and validate the password/user name combination.  If you look at the project tracker Principal and Identity classes, you will see the example on how to authenticate against SQL server.  That is the code you need to replace with ADAM calls.

sash_kr posted on Saturday, October 19, 2013

I would like to translate CSLA on Ukrainian language. How could I achieve that?

RockfordLhotka replied on Saturday, October 19, 2013

Take the resources.resx file (US English) and make a copy so the file name includes your language code, then translate all the text to your language. You can look at the many other resource files that have been translated to see how this works.

Then you can send me the resulting file, or issue a pull request in GitHub., or post it here on the forum.

Thank you for adding a language to the framework!

In the Rolodex example, business classes are "mirrored" in the client code from the server code in a way that is much like a shortcut. How does one set this up in the IDE?

sergeyb replied on Friday, December 26, 2008

Right-click on client’s project’s destination folder, select add existing item, browse to server’s class, you will see down arrow on OK button, click on that down arrow, select add as link.

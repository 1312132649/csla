I just put an update of the demo code for the CSLA .NET for Silverlight video series online. This update is for verison 3.8.1.

The changes are relatively minor:

  • Fix the PropertyStatus elements by removing all Source values, changing Property values to binding expressions and referencing target controls with the TargetControl value
  • Fix the InvokeMethod elements by removing the TargetResource value and making sure the DataContext points to the correct resource object

I just put a refresh of the 3.6.1 test release online - both Windows and Silverlight versions.

Consult the change logs and issue tracker database for more details, but this refresh includes:
  • Fix several bugs reported on this forum, and by others
  • Eliminates the need for _forceInit (with some restrictions in Silverlight)
  • Adds LoadProperty() method to ObjectFactory base class
If you are using 3.6.0, please grab this version and give it a go. Let me know if you encounter any issues. I am projecting a final release on 1/30 or 2/2 - barring any issues with this release.

Philip posted on Monday, July 13, 2009

I am glad to announce that the Beta version of Code Complete 4.0 is now available for download for subscribers. The major new feature is the support for Silverlight. On top of generating fully working Web, Win and WPF applications already supported in previous versions, you can now generate fully working Silverlight applications based on both SL2 (with CSLA.NET 3.6.x)  and SL3 (with CSLA.NET 3.7).  The new version also uses the Microsoft Enterprise Library 4.1.

The Nightly Build 2011.12.05 generates Encapsulated Invoke. At this stage the generated code uses IDataReader and parameter passing. DTOs are scheduled for stage 2.

New Features (since Nightly Build 2011.10.05):

  • Global project validation
  • Encapsulated Invoke model (using IDataReader and parameter passing)
  • Generate Criteria as CriteriaBase
  • Generate Criteria as BusinessBase
  • Generate non nested public Criteria class
  • Allow direct calls to Child_XYZ methods, droping generation of internal factory methods
  • Allow child update order

Plus lots of improvements and fixes.

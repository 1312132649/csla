RockfordLhotka replied on Monday, February 12, 2007

Make sure the CslaAuthentication value in <appSettings> is the same in both client and server config files.

RockfordLhotka replied on Monday, February 12, 2007

Oh, and make sure you either logout before trying to use the data portal. If you look at chapters 8, 9 or 10 you'll notice that in all cases the application does a logout before anything else happens - effectively making sure that the current principal on the client is an instance of your custom principal object.

tiago posted on Saturday, March 30, 2013

SafeDataReader has a GetDateTime that returns a DateTime. This fits the following System.Data.DbType:

  • Date
  • DateTime
  • DateTime2
  • SmallDateTime

SafeDataReader is missing the following methods:

  • GetDateTimeOffset that should return a DateTimeOffset
  • GetTime that should return a TimeSpan

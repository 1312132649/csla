RockfordLhotka replied on Wednesday, January 26, 2011

The DataObjectField attribute applies only to ASP.NET Web Forms, and even there its utility is limited. If you are building a Web Forms UI you might still use this attribute if you find it helpful.

Since Web Forms is now just one of many UI types I've stopped putting that attribute into my samples, as it just causes confusion and clutter for most people.

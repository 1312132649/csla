RockfordLhotka replied on Saturday, March 30, 2013

At a minimum you'll need:

  • Windows 8
  • Visual Studio 2012 with Update 1
  • Latest nuget extension for VS12
  • Windows Phone SDK
  • Silverlight 5 SDK (I think this comes with VS12?)
  • ASP.NET MVC 3
  • ASP.NET MVC 4
  • Entity Framework 4
  • Entity Framework 5

Some of those (like EF) should be automatic via nuget. Certainly the async targeting pack comes via nuget.
